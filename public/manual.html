📘 Workdog Archive (for Inhee) 사용 매뉴얼

버전 1.1 · 실사용 가이드

← 메인으로

1) 기본 흐름

  • 메인 화면 우측 아이콘 ＋ 으로 폴더를 생성합니다.
  • 폴더명을 클릭하면 폴더 상세로 이동합니다.
  • 문서 목록 상단의 Upload 버튼으로 업로드합니다.
  • 업로드 후 자동으로 텍스트 추출/요약/분류가 실행됩니다.

2) 폴더 관리

  • 생성: 모달 팝업에서 폴더명/색상/설명 입력 후 생성
  • 수정: 폴더 카드의 ✎ 버튼 (생성과 동일한 구조의 수정 팝업)
  • 삭제: 폴더 카드의 ✕ 버튼

폴더 색상은 폴더 아이콘 및 폴더명에 동일하게 반영됩니다.

3) 문서 업로드

지원 형식: hwp pdf xlsx xls txt

  • 문서 목록 헤더의 Upload 버튼 클릭
  • 업로드 팝업에서 문서 제목(선택) + 파일 선택 후 업로드
  • 처리 실패 시 상세 팝업에서 에러 원인을 확인

4) 검색 / 필터 / 정렬

필터

  • 제목 검색
  • 카테고리 드롭다운 필터
  • 파일형식 드롭다운 필터
  • 태그 포함 검색
  • 중요문서만 보기

정렬

  • 최신순
  • 오래된순
  • 제목순
  • 중요문서 우선

필터 접기/펼치기는 꺽쇠 아이콘 버튼(⌄ / ⌃)으로 제어합니다.

5) 문서 목록 읽는 법

  • 중요: 회색 별(☆) / 노란 별(★)
  • 요약: 길면 ... 처리, 마우스 올리면 전체 표시
  • 메모: Y/N로 표시, 마우스 올리면 메모 내용 표시
  • 업로드일: 표에는 날짜(yyyy-mm-dd), 툴팁에 시분초

6) 문서 상세/메모/복사

  • 목록의 ☰ 버튼으로 문서 상세 팝업 열기
  • 요약결과 복사 버튼으로 한줄요약/핵심포인트/카테고리/태그 일괄 복사
  • 문서 메모 작성 후 저장

7) 운영 팁

  • 자주 보는 문서는 ★ 표시 후 정렬을 중요문서 우선으로 사용하세요.
  • 메모 컬럼(Y/N)을 활용하면 후속 확인 문서를 빠르게 찾을 수 있습니다.
  • 에러 문서는 상세 팝업에서 추출/AI 에러를 먼저 확인하세요.
